# Summit Challenge

## Objective
Apply SOC investigation, alert triage, threat analysis, and incident response concepts learned throughout the SOC Level 1 path in a practical challenge-based environment.

---

## Concepts Covered
- SOC investigations
- Alert triage
- Incident response
- Threat analysis
- SIEM investigations
- Log analysis
- Security monitoring

---

## Challenge Overview

The Summit Challenge simulated real-world SOC scenarios where multiple alerts and suspicious activities required investigation and analysis.

The challenge focused on:
- Investigating security events
- Identifying malicious activity
- Prioritizing alerts
- Applying defensive security concepts
- Responding to incidents effectively

---

## Investigation Workflow

### Initial Alert Review
Performed initial triage by:
- Reviewing security alerts
- Identifying affected systems
- Gathering contextual information
- Determining alert severity

---

### Event Analysis
Investigated logs and security events to:
- Identify suspicious activity
- Correlate related events
- Analyze Indicators of Compromise (IOCs)
- Validate potential threats

---

### Threat Identification
Analyzed:
- Suspicious IP addresses
- Unusual authentication activity
- Security alerts
- Potential malicious behavior

---

## SOC Skills Applied

The challenge involved practical use of:
- Alert triage workflows
- SIEM investigations
- Incident documentation
- Threat intelligence concepts
- Security analysis techniques
