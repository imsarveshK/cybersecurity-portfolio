# Introduction to SIEM

## Objective
Understand the fundamentals of Security Information and Event Management (SIEM) systems, including log collection, event correlation, alert generation, and security monitoring within a Security Operations Center (SOC).

---

## Concepts Covered
- Security Information and Event Management (SIEM)
- Log collection
- Event correlation
- Security monitoring
- Alert generation
- Incident investigation

---

## SIEM Overview

A Security Information and Event Management (SIEM) platform is used to:
- Collect logs from multiple sources
- Analyze security events
- Correlate suspicious activity
- Generate security alerts
- Support incident investigations

SIEM solutions provide centralized visibility across organizational infrastructure.

---

## Purpose of SIEM

SOC teams use SIEM platforms to:
- Detect malicious activity
- Monitor security events
- Investigate incidents
- Improve threat visibility
- Support compliance requirements

---

## Log Sources

SIEM platforms collect logs from:
- Endpoints
- Servers
- Firewalls
- Network devices
- Applications
- Authentication systems

These logs are aggregated into a centralized platform for analysis.

---

## Event Correlation

SIEM systems correlate events from multiple sources to:
- Identify attack patterns
- Detect suspicious behavior
- Reduce false positives
- Improve alert accuracy

Correlation rules help analysts prioritize threats efficiently.

---

## Alert Generation

SIEM platforms generate alerts based on:
- Detection rules
- Suspicious activity
- Known Indicators of Compromise (IOCs)
- Behavioral anomalies

SOC analysts investigate these alerts during incident response operations.

---

## Common SIEM Capabilities

Typical SIEM features include:
- Centralized log management
- Real-time monitoring
- Alerting and notifications
- Dashboard visualization
- Search and investigation tools
- Reporting and compliance support

---

## SIEM Investigation Workflow

### Typical Investigation Process
- Review alert details
- Analyze related logs
- Correlate events
- Validate suspicious activity
- Escalate confirmed incidents
