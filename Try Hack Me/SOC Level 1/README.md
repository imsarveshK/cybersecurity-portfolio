# SOC Level 1 Path

## Overview
This repository documents my progress through the TryHackMe SOC Level 1 learning path.  
The path focuses on defensive cybersecurity concepts including threat detection, log analysis, SIEM investigation, network monitoring, and incident response fundamentals.

The goal of this path is to build practical blue team skills used in real-world Security Operations Centers (SOC).

- Incident detection and triage

- Windows event log analysis
