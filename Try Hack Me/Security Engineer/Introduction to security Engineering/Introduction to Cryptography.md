## Cryptography Overview

Cryptography is the practice of securing information by transforming data into unreadable formats to protect:
- Confidentiality
- Integrity
- Authenticity

Cryptography is widely used to secure:
- Communications
- Authentication systems
- Sensitive data
- Online transactions

---

## Encryption & Decryption

### Encryption
The process of converting plaintext into ciphertext using an encryption algorithm and key.

Purpose:
- Protect sensitive information from unauthorized access

---

### Decryption
The process of converting ciphertext back into readable plaintext using the correct key.

---

## Symmetric Encryption

Symmetric encryption uses the same key for:
- Encryption
- Decryption

Characteristics:
- Fast performance
- Efficient for large data encryption

Examples:
- AES
- DES

---

## Asymmetric Encryption

Asymmetric encryption uses:
- Public key
- Private key

Characteristics:
- Secure key exchange
- Supports digital signatures

Examples:
- RSA
- ECC

---

## Hashing

Hashing converts data into a fixed-length hash value.

Characteristics:
- One-way process
- Cannot be reversed easily
- Used for integrity verification

Examples:
- MD5
- SHA1
- SHA256

## Digital Signatures

Digital signatures help:
- Verify authenticity
- Ensure integrity
- Support non-repudiation

They are commonly used in:
- Secure communications
- Software signing
- Authentication systems

---

## Cryptography in Security

Cryptography is used in:
- HTTPS/TLS
- VPNs
- Password storage
- Secure authentication
- Digital certificates

---

## Common Cryptographic Risks

Potential weaknesses include:
- Weak algorithms
- Poor key management
- Outdated protocols
- Improper implementations

## Best Practices

Recommended approaches:
- Use strong encryption algorithms
- Protect cryptographic keys securely
- Avoid outdated hashing algorithms
- Implement secure communication protocols
- Regularly update cryptographic implementations

- Symmetric and asymmetric encryption serve different purposes
- Hashing supports integrity verification
- Digital signatures improve authenticity and trust
