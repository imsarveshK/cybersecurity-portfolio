# Eviction Challenge

## Objective
Apply SOC investigation and incident response skills to analyze suspicious activity, identify malicious behavior, and investigate security events within a simulated SOC environment.

---

## Concepts Covered
- Alert triage
- Incident investigation
- Log analysis
- Threat detection
- Indicators of Compromise (IOCs)
- Security monitoring

---

## Challenge Overview

The Eviction Challenge simulated a real-world SOC investigation scenario involving suspicious activity within an environment requiring analysis and response.

The challenge focused on:
- Investigating alerts
- Reviewing security logs
- Identifying malicious indicators
- Correlating suspicious events
- Applying defensive investigation techniques

---

## Investigation Workflow

### Initial Alert Review
Performed initial triage by:
- Reviewing alert information
- Identifying affected systems
- Gathering contextual details
- Assessing alert severity

---

### Log & Event Analysis
Analyzed available logs and events to:
- Identify suspicious behavior
- Correlate related activity
- Detect Indicators of Compromise (IOCs)
- Investigate attacker actions

---

### Threat Investigation
Focused on identifying:
- Malicious IP addresses
- Suspicious processes
- Unauthorized access attempts
- Potential persistence activity

---

## SOC Skills Applied

The challenge involved:
- Security event analysis
- Alert prioritization
- Threat investigation
- IOC identification
- Incident response decision-making
