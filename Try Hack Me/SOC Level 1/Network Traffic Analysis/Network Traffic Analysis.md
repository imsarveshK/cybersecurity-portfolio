# Network Traffic Basics

## Objective
Understand the fundamentals of network traffic analysis and learn how SOC analysts monitor, inspect, and investigate network communications to identify suspicious or malicious activity.

---

## Concepts Covered
- Network traffic analysis
- Packet inspection
- Protocol analysis
- Network monitoring
- Suspicious traffic detection
- Indicators of Compromise (IOCs)

---

## Network Traffic Analysis Overview

Network Traffic Analysis (NTA) involves monitoring and analyzing network communications to:
- Detect malicious activity
- Identify suspicious behavior
- Investigate security incidents
- Monitor network health

SOC analysts use traffic analysis to gain visibility into network activity and identify potential threats.

---

## Common Network Protocols

The room covered several important protocols including:
- TCP
- UDP
- HTTP
- HTTPS
- DNS
- ICMP

Understanding protocol behavior helps analysts identify abnormal or malicious traffic patterns.

---

## Packet Inspection

Packet analysis involves examining:
- Source and destination IP addresses
- Ports and protocols
- Payload data
- Connection behavior

This helps analysts investigate suspicious communications and potential attacks.

---

## Suspicious Network Activity

Examples of suspicious activity may include:
- Unusual outbound traffic
- Repeated failed connections
- DNS anomalies
- Beaconing behavior
- Large data transfers
- Connections to malicious IP addresses

---

## Indicators of Compromise (IOCs)

Common network-based IOCs:
- Suspicious IP addresses
- Malicious domains
- Unusual traffic spikes
- Unauthorized connections
- Suspicious DNS requests

---

## SOC Investigation Workflow

Typical network investigation process:
- Capture or review traffic
- Filter suspicious activity
- Analyze protocols and connections
- Identify malicious indicators
- Escalate confirmed threats

## Best Practices

Recommended approaches:
- Monitor network traffic continuously
- Investigate unusual communication patterns
- Correlate network events with alerts
- Use protocol filtering effectively
- Validate suspicious indicators carefully

---

## Key Learnings
- Network traffic analysis improves visibility into attacks
- Protocol understanding is essential for investigations
- Suspicious communication patterns can indicate compromise
- Packet inspection helps identify malicious activity
- SOC analysts use traffic analysis during incident investigations
